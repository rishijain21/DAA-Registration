## 🎯 Overview

Built for the **Directorate of Alumni Affairs (DAA), SRM Institute of Science and Technology**, this project is a registration landing page for **Global Meet 3.0** — an alumni-student networking event.

The page allows both **students and alumni** to register for the event. Form submissions are collected and stored directly in a spreadsheet, making data management simple for the event organizers without needing a backend server.

---

## ✨ Features

- **Dual Registration** — separate flows for Students and Alumni
- **Animated UI** — smooth CSS animations for a polished event-page feel
- **Confirmation Page** — dedicated `thanks.html` shown after successful submission
- **Fully Responsive** — works across desktop and mobile browsers
- **No Backend Required** — form data stored directly in a spreadsheet via form integration
- **Deployed on GitHub Pages** — zero-cost static hosting, live instantly

## 🚀 Running Locally

No build step needed — it's plain HTML/CSS.

```bash
git clone https://github.com/rishijain21/DAA-Registration.git
cd DAA-Registration
```

Open `index.html` directly in your browser, or use a local server:

```bash
# Using VS Code Live Server extension, or:
npx serve .
```
